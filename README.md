Hyper
=============
Hyper scss file structure system.

````
|– tools/

|   |– _list.scss
|   |– _grid.scss

|   …                     
|
|– base/
|   |– __all.scss
|   |– _normalize.scss      
|   |– _fonts.scss     
|   |– _typography.scss
|   |– _icons.scss   
|   |– _reboot.scss            
|   …  
|
|– elements/
|   |– __all.scss
|   |– _links.scss      
|   |– _hr.scss             
|   …  
|                   
|– objects/
|   |– __all.scss
|   |– _header.scss
|   |– _main-nav.scss

|   …

|   |– _modal.scss

|   …                     
|
– hyper.scss  # Hyper Main Sass file
````
